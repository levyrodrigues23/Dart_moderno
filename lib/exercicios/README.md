# 📚 Exercícios de Dart - Baseados no dart_fundamentos

Bem-vindo à pasta de exercícios! Aqui você vai praticar TODOS os conceitos que aprendeu.

## Estrutura dos Exercícios

### Fundamentos
1. **exercicio_null_safety.dart** - Nullable, operadores !, ??, ?., late variables
2. **exercicio_colecoes.dart** - List, Set, Map, where, map, fold, reduce

### POO Básico
3. **exercicio_classes.dart** - Classes, construtores (padrão, nomeados, factory), encapsulamento, getter/setter
4. **exercicio_records.dart** - Records posicionais, nomeados, destructuring

### POO Avançado
5. **exercicio_interfaces.dart** - Abstract class, abstract interface class, interface class, polimorfismo
6. **exercicio_mixins.dart** - Mixins, restrições com 'on', mixin class, contratos
7. **exercicio_extensions.dart** - Extension methods em tipos existentes

### Recursos Especiais
8. **exercicio_operadores_callable.dart** - Operadores is/as, type check, type cast, callable class
9. **exercicio_modificadores.dart** - Base, final, sealed classes, pattern matching

### Desafio
10. **desafio_final.dart** - Mini e-commerce combinando TODOS os conceitos!

## Como fazer

- Cada arquivo tem instruções nos comentários
- Complete o código onde está marcado com `// TODO:`
- Execute com: `dart run lib/exercicios/nome_do_arquivo.dart`
- As respostas estão comentadas no final de cada exercício

## Ordem sugerida de estudo

```
1. Null Safety → 2. Coleções → 3. Classes → 4. Records
→ 5. Interfaces → 6. Mixins → 7. Extensions
→ 8. Operadores/Callable → 9. Modificadores → 10. Desafio Final
```

**Dica:** Tente fazer sem olhar as respostas primeiro! 💪
